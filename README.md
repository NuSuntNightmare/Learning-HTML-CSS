# Learning-HTML-CSS 📚

A comprehensive learning repository containing **75.3% HTML** and **24.7% CSS** dedicated to mastering HTML and CSS fundamentals with organized, practical examples.

## 📁 Repository Structure

### HTML & CSS Folder

This repository is organized into the main `HTML & CSS` folder which contains 40+ subfolders, each focusing on specific HTML and CSS concepts and components:

#### **Layout & Positioning**
- **Flexbox** - Flexible box layout model for responsive designs
- **Grid** - CSS Grid system for creating complex layouts
- **Position** - Positioning strategies (static, relative, absolute, fixed)
- **Float** - Traditional floating layout techniques
- **Display** - Display properties and behaviors
- **Width & Height** - Sizing elements with CSS
- **Margin** - Margin spacing and alignment
- **Overflow** - Handling content overflow behavior

#### **Styling & Appearance**
- **Borders** - Border styles, colors, and properties
- **Shadow** - Box shadows and text shadows for depth effects
- **Transform** - CSS transforms for scaling, rotating, and skewing
- **Animation** - CSS animations and transitions for motion
- **Pseudo** - Pseudo-classes and pseudo-elements (:hover, :active, ::before, ::after, etc.)
- **Font Text** - Font properties, sizes, and text styling
- **Fonts Icon** - Icon fonts and custom font implementations
- **Resolution** - Responsive design and media queries

#### **HTML Elements & Components**
- **Elements** - Basic HTML elements and their usage
- **Button** - Styled buttons and button variations
- **Forms** - Form elements and input styling
- **Tables** - Table structure and styling
- **List** - Ordered, unordered, and description lists
- **Span & Div** - Inline and block container elements
- **Formatting** - Text formatting and semantic HTML

#### **Content & Media**
- **Video** - HTML5 video embedding and controls
- **Audio** - HTML5 audio embedding and controls
- **img** - Image elements and image styling

#### **Interactive Components**
- **Navigation** - Navigation bars and menu structures
- **Menu (drop down)** - Dropdown menus and navigation dropdowns
- **Header & Footer** - Semantic header and footer structures
- **Interactive - Img Gallery** - Image galleries and lightboxes

#### **Advanced Topics**
- **Combinators** - CSS selectors and combinators (descendant, child, adjacent, sibling)
- **Favicon** - Website favicon setup and implementation
- **Layout SEO** - Semantic HTML and SEO optimization
- **Preview** - Preview components and examples
- **Lyrics** - Example content structure (music lyrics display)
- **Playwrite_DE_SAS** - Font file resources
- **CSS** - Core CSS concepts and standalone CSS examples
- **Paginations** - Pagination component styling

#### **Media Resources**
- **mp4** - Video files for examples
- **img** - Image files for demonstrations
